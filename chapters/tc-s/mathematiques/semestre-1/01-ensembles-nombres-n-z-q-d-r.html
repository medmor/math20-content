Les ensembles de nombres N, Z, Q, D et R

1. Les nombres naturels (N)

Définition : L'ensemble des nombres naturels comprend tous les entiers positifs à partir de 1.

Notation :

N = {1, 2, 3, 4, 5, ...}

Propriétés :

  • Fermé pour l'addition et la multiplication
  • Non fermé pour la soustraction et la division
  • Possède un plus petit élément : 1

Exemples :

  • 7 ∈ N
  • 100 ∈ N
  • 0 ∉ N
  • -3 ∉ N

2. Les nombres entiers naturels (W) - N0

Définition : L'ensemble des nombres entiers naturels comprend tous les nombres naturels et zéro.

Notation :

W = {0, 1, 2, 3, 4, 5, ...}

Propriétés :

  • Fermé pour l'addition et la multiplication
  • Non fermé pour la soustraction et la division
  • Possède un plus petit élément : 0

Exemples :

  • 0 ∈ W
  • 15 ∈ W
  • -1 ∉ W
  • 2.5 ∉ W

3. Les nombres entiers relatifs (Z)

Définition : L'ensemble des nombres entiers relatifs comprend tous les nombres entiers naturels et leurs opposés (négatifs).

Notation :

Z = {..., -3, -2, -1, 0, 1, 2, 3, ...}

Propriétés :

  • Fermé pour l'addition, la soustraction et la multiplication
  • Non fermé pour la division
  • N'a pas de plus petit ni de plus grand élément

Sous-ensembles :

  • Z+ = {1, 2, 3, ...} (entiers positifs)
  • Z- = {..., -3, -2, -1} (entiers négatifs)
  • Z0+ = {0, 1, 2, 3, ...} (entiers non négatifs)

Exemples :

  • -8 ∈ Z
  • 0 ∈ Z
  • 42 ∈ Z
  • ½ ∉ Z
  • 3.14 ∉ Z

4. Les nombres décimaux (D)

Définition : L'ensemble des nombres décimaux comprend tous les nombres qui peuvent s'écrire sous la forme a/10n où a ∈ Z et n ∈ N.

Notation :

D = {a/10n | a ∈ Z, n ∈ N}

Propriétés :

  • Fermé pour l'addition, la soustraction et la multiplication
  • Non fermé pour la division

Exemples :

  • 0.5 = 5/10 ∈ D
  • -3.75 = -375/100 ∈ D
  • 7 = 7/1 ∈ D
  • 0.333... ∉ D (car 1/3 n'est pas décimal)

5. Les nombres rationnels (Q)

Définition : Un nombre rationnel est tout nombre qui peut s'écrire sous la forme a/b où a ∈ Z et b ∈ Z\{0}.

Notation :

Q = {a/b | a ∈ Z, b ∈ Z, b ≠ 0}

Propriétés :

  • Fermé pour l'addition, la soustraction, la multiplication et la division (sauf par 0)
  • Peut s'écrire sous forme décimale finie ou périodique
  • Dense dans R (entre deux rationnels, il existe toujours un autre rationnel)

Exemples :

  • 3/4 ∈ Q
  • -2 = -2/1 ∈ Q
  • 0.75 = 3/4 ∈ Q
  • 0.333... = 1/3 ∈ Q
  • √2 ∉ Q
  • π ∉ Q

6. Les nombres irrationnels (I)

Définition : Un nombre irrationnel est un nombre réel qui ne peut pas s'écrire sous la forme a/b avec a, b ∈ Z et b ≠ 0. Son développement décimal est non fini et non périodique.

Propriétés :

  • Non fermé pour aucune opération (la somme ou le produit peut être rationnel ou irrationnel)
  • Ne peut pas s'écrire sous forme de fraction

Exemples :

  • √2 ≈ 1.41421356... ∈ I
  • π ≈ 3.14159265... ∈ I
  • e ≈ 2.71828182... ∈ I
  • √3 ∈ I
  • 1/3 ∉ I (c'est rationnel)

7. Les nombres réels (R)

Définition : L'ensemble des nombres réels comprend tous les nombres rationnels et irrationnels.

Notation :

R = Q ∪ I

Propriétés :

  • Corps ordonné complet
  • Chaque point sur la droite numérique correspond à un nombre réel
  • Fermé pour l'addition, la soustraction, la multiplication et la division (sauf par 0)

Exemples :

  • 5 ∈ R
  • -3/2 ∈ R
  • √7 ∈ R
  • π ∈ R
  • i = √-1 ∉ R (nombre imaginaire)

8. Relations entre les ensembles

Inclusions :

N ⊂ W ⊂ Z ⊂ D ⊂ Q ⊂ R

I ⊂ R et I ∩ Q = ∅

Schéma :

R (nombres réels)
├── Q (nombres rationnels)
│   ├── D (nombres décimaux)
│   │   ├── Z (entiers relatifs)
│   │   │   ├── W (entiers naturels)
│   │   │   │   └── N (naturels)
│   │   │   └── entiers négatifs
│   │   └── décimaux non entiers
│   └── rationnels non décimaux
└── I (nombres irrationnels)

Exercices

Exercice 1

Classer chaque nombre dans les ensembles auxquels il appartient (N, W, Z, D, Q, I, R) :

  • a) 0
  • b) -7
  • c) 2.333...
  • d) √9
  • e) √7
  • f) 1/4
  • g) -0.5
  • h) 1000
  • i) π

Exercice 2

Vrai ou Faux :

  • a) Tous les entiers sont des nombres rationnels.
  • b) Tous les nombres rationnels sont des entiers.
  • c) 0 est un nombre naturel.
  • d) La somme de deux nombres irrationnels est toujours irrationnelle.
  • e) Tout nombre réel est soit rationnel, soit irrationnel.
  • f) √16 est un nombre irrationnel.

Exercice 3

Donner un exemple de :

  • a) Un nombre rationnel qui n'est pas un entier.
  • b) Un nombre réel qui n'est pas rationnel.
  • c) Un nombre entier qui n'est pas naturel.
  • d) Un nombre décimal qui n'est pas entier.

Exercice 4

Placer les nombres suivants sur une droite numérique :

-3, 0, 1/2, √2, -1.5, π

Exercice 5

Démontrer que √2 est irrationnel. (Indication : utiliser un raisonnement par l'absurde.)
