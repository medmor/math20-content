Calcul vectoriel dans le plan

1. Vecteurs

Définition : Vecteur

Un vecteur dans le plan est un couple ordonné de nombres réels, noté :

$$\vec{v} = (v_1, v_2) \quad \text{ou} \quad \vec{v} = \begin{pmatrix} v_1 \\ v_2 \end{pmatrix}$$

où $v_1$ et $v_2$ sont les composantes du vecteur.

Interprétation géométrique

Un vecteur peut être représenté comme un segment orienté d'un point initial $A(x_1, y_1)$ à un point final $B(x_2, y_2)$, avec :

$$\vec{AB} = (x_2 - x_1, y_2 - y_1)$$

Vecteur nul

$$\vec{0} = (0, 0)$$

Opposé d'un vecteur

$$-\vec{v} = (-v_1, -v_2)$$

2. Opérations sur les vecteurs

Addition de vecteurs

Pour deux vecteurs $\vec{u} = (u_1, u_2)$ et $\vec{v} = (v_1, v_2)$ :

$$\vec{u} + \vec{v} = (u_1 + v_1, u_2 + v_2)$$

Propriétés de l'addition

  • Commutativité : $\vec{u} + \vec{v} = \vec{v} + \vec{u}$
  • Associativité : $(\vec{u} + \vec{v}) + \vec{w} = \vec{u} + (\vec{v} + \vec{w})$
  • Élément neutre : $\vec{u} + \vec{0} = \vec{u}$
  • Opposé : $\vec{u} + (-\vec{u}) = \vec{0}$

Multiplication par un scalaire

Pour un scalaire $\lambda \in \mathbb{R}$ et un vecteur $\vec{v} = (v_1, v_2)$ :

$$\lambda \vec{v} = (\lambda v_1, \lambda v_2)$$

Propriétés

  • $\lambda(\vec{u} + \vec{v}) = \lambda\vec{u} + \lambda\vec{v}$
  • $(\lambda + \mu)\vec{v} = \lambda\vec{v} + \mu\vec{v}$
  • $(\lambda \mu)\vec{v} = \lambda(\mu\vec{v})$
  • $1 \cdot \vec{v} = \vec{v}$

3. Norme d'un vecteur

Définition

La norme (ou longueur) d'un vecteur $\vec{v} = (v_1, v_2)$ est :

$$\|\vec{v}\| = \sqrt{v_1^2 + v_2^2}$$

Propriétés

  • $\|\vec{v}\| \geq 0$ et $\|\vec{v}\| = 0 \Leftrightarrow \vec{v} = \vec{0}$
  • $\|\lambda\vec{v}\| = |\lambda| \cdot \|\vec{v}\|$
  • Inégalité triangulaire : $\|\vec{u} + \vec{v}\| \leq \|\vec{u}\| + \|\vec{v}\|$

4. Vecteur unitaire

Définition

Un vecteur unitaire est un vecteur de norme 1.

Pour tout vecteur non nul $\vec{v}$, le vecteur unitaire dans la direction de $\vec{v}$ est :

$$\hat{v} = \frac{\vec{v}}{\|\vec{v}\|} = \left(\frac{v_1}{\|\vec{v}\|}, \frac{v_2}{\|\vec{v}\|}\right)$$

5. Colinéarité

Définition

Deux vecteurs $\vec{u}$ et $\vec{v}$ sont colinéaires s'il existe un réel $\lambda$ tel que :

$$\vec{u} = \lambda\vec{v} \quad \text{ou} \quad \vec{v} = \lambda\vec{u}$$

Condition de colinéarité

Deux vecteurs $\vec{u} = (u_1, u_2)$ et $\vec{v} = (v_1, v_2)$ sont colinéaires si et seulement si :

$$u_1 \cdot v_2 - u_2 \cdot v_1 = 0$$

c'est-à-dire que le déterminant est nul.

Vecteurs de même direction

  • Si $\lambda > 0$ : les vecteurs ont le même sens
  • Si $\lambda < 0$ : les vecteurs ont des sens opposés

6. Base du plan

Définition

Une base du plan est un couple de deux vecteurs non colinéaires $(\vec{i}, \vec{j})$.

Tout vecteur $\vec{v}$ du plan peut s'écrire de manière unique :

$$\vec{v} = x\vec{i} + y\vec{j}$$

où $(x, y)$ sont les coordonnées de $\vec{v}$ dans cette base.

Base orthonormée

Une base $(\vec{i}, \vec{j})$ est orthonormée si :

  • $\|\vec{i}\| = \|\vec{j}\| = 1$
  • $\vec{i}$ et $\vec{j}$ sont orthogonaux

Exercices

Exercice 1

Soient $\vec{u} = (2, 3)$ et $\vec{v} = (-1, 4)$. Calculer :

  • $\vec{u} + \vec{v}$
  • $2\vec{u} - 3\vec{v}$
  • $\|\vec{u}\|$ et $\|\vec{v}\|$

Exercice 2

Déterminer si les vecteurs $\vec{u} = (2, -3)$ et $\vec{v} = (4, -6)$ sont colinéaires.

Exercice 3

Soit $\vec{u} = (3, 4)$. Calculer le vecteur unitaire dans la direction de $\vec{u}$.

Exercice 4

Soient $A(1, 2)$ et $B(4, 6)$. Calculer les coordonnées du vecteur $\vec{AB}$ et sa norme.

Exercice 5

Vérifier que les vecteurs $\vec{u} = (1, 0)$ et $\vec{v} = (0, 1)$ forment une base orthonormée du plan.
