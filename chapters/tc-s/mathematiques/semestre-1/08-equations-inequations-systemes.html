2-3/ Détermination des solutions de l'équation ax² + bx + c (a ≠ 0)

Soit l'équation ax² + bx + c = 0, et son discriminant Δ = b² - 4ac.

  • Si Δ > 0, l'équation admet deux solutions distinctes (ou deux racines distinctes) dans ℝ : x₁ = (-b + √Δ)/(2a) et x₂ = (-b - √Δ)/(2a).

  • Si Δ = 0, l'équation admet une solution (solution double) dans ℝ : x₀ = -b/(2a).

  • Si Δ < 0, l'équation n'a pas de solution dans ℝ : S = ∅.

2-4/ La somme et le produit des racines de l'équation ax² + bx + c (a ≠ 0)

Si l'équation admet deux racines distinctes x₁ et x₂, alors on a :

x₁ + x₂ = -b/a et x₁ × x₂ = c/a

Factorisation du trinôme de second degré ax² + bx + c (a ≠ 0)

Δ est le discriminant de l'équation x ∈ ℝ / ax² + bx + c .

  • Si Δ < 0, l'équation n'a pas de solution dans ℝ, on ne peut pas factoriser ax² + bx + c sous forme de produit de deux polynômes de 1er degré (deux monômes).

Équations, inéquations et systèmes

Dans ce chapitre, nous allons étudier les équations, les inéquations et les systèmes d’équations du premier degré à une ou deux inconnues. Nous verrons également comment résoudre des équations du second degré.

1. Équations du premier degré à une inconnue

Une équation du premier degré à une inconnue est une égalité de la forme ax + b = 0, où a et b sont des nombres réels donnés, avec a ≠ 0.

Sa solution est : x = -b/a

2. Inéquations du premier degré à une inconnue

Une inéquation du premier degré à une inconnue est une inégalité de la forme ax + b > 0 (ou ≥, <, ≤), avec a ≠ 0.

Pour la résoudre, on isole l’inconnue x en appliquant les règles de transformation des inégalités.

3. Systèmes d’équations linéaires à deux inconnues

Un système de deux équations linéaires à deux inconnues x et y s’écrit sous la forme :

  • a₁x + b₁y = c₁
  • a₂x + b₂y = c₂

On peut le résoudre par substitution, par combinaison linéaire ou graphiquement.

4. Équations du second degré

Une équation du second degré à une inconnue est de la forme : ax² + bx + c = 0, avec a ≠ 0.

Le discriminant est : Δ = b² - 4ac

  • Si Δ > 0, l’équation admet deux solutions réelles distinctes : x₁ = (-b - √Δ)/(2a) et x₂ = (-b + √Δ)/(2a)
  • Si Δ = 0, l’équation admet une solution double : x = -b/(2a)
  • Si Δ < 0, l’équation n’admet aucune solution réelle.

Résoudre le système :

  • 2x + y = 5
  • x - y = 1

Solution : En ajoutant les deux équations, on obtient : 3x = 6 ⇒ x = 2. En remplaçant dans la deuxième équation : 2 - y = 1 ⇒ y = 1. La solution est donc (2 ; 1).
