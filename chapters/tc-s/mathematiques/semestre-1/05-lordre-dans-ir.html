L'ordre dans IR

1. Inégalités dans IR

Définitions

Pour $a, b \in \mathbb{R}$ :

  • $a < b$ signifie que $a$ est strictement inférieur à $b$
  • $a \leq b$ signifie que $a$ est inférieur ou égal à $b$
  • $a > b$ signifie que $a$ est strictement supérieur à $b$
  • $a \geq b$ signifie que $a$ est supérieur ou égal à $b$

Propriétés de la relation d'ordre

  1. Réflexivité : $a \leq a$
  2. Antisymétrie : Si $a \leq b$ et $b \leq a$, alors $a = b$
  3. Transitivité : Si $a \leq b$ et $b \leq c$, alors $a \leq c$
  4. Totalité : Pour tous $a, b \in \mathbb{R}$, $a \leq b$ ou $b \leq a$

2. Opérations et inégalités

Addition

Si $a \leq b$ et $c \in \mathbb{R}$, alors :

$$a + c \leq b + c$$

Multiplication

  • Si $a \leq b$ et $c > 0$, alors $ac \leq bc$
  • Si $a \leq b$ et $c < 0$, alors $ac \geq bc$ (le sens change)

Inversion

  • Si $0 < a \leq b$, alors $\frac{1}{a} \geq \frac{1}{b}$
  • Si $a \leq b < 0$, alors $\frac{1}{a} \geq \frac{1}{b}$

3. Intervalles

Définition

Un intervalle est une partie de $\mathbb{R}$ telle que, pour tout couple de points de cette partie, tous les points intermédiaires lui appartiennent.

Types d'intervalles

NotationEnsembleDescription
$[a, b]$$\{x \in \mathbb{R} \mid a \leq x \leq b\}$Fermé
$]a, b[$$\{x \in \mathbb{R} \mid a < x < b\}$Ouvert
$[a, b[$$\{x \in \mathbb{R} \mid a \leq x < b\}$Semi-ouvert
$]a, b]$$\{x \in \mathbb{R} \mid a < x \leq b\}$Semi-ouvert
$[a, +\infty[$$\{x \in \mathbb{R} \mid x \geq a\}$Non borné
$]-\infty, a]$$\{x \in \mathbb{R} \mid x \leq a\}$Non borné

Propriétés des intervalles

  • L'intersection de deux intervalles est un intervalle
  • L'union de deux intervalles n'est pas toujours un intervalle

4. Valeur absolue

Définition

La valeur absolue d'un réel $x$ est :

$$|x| = \begin{cases} x & \text{si } x \geq 0 \\ -x & \text{si } x < 0 \end{cases}$$

Propriétés

  • $|x| \geq 0$ et $|x| = 0 \Leftrightarrow x = 0$
  • $|-x| = |x|$
  • $|xy| = |x| \cdot |y|$
  • $|x + y| \leq |x| + |y|$ (Inégalité triangulaire)
  • $|x - y| \geq ||x| - |y||$

Distance dans IR

La distance entre deux réels $x$ et $y$ est :

$$d(x, y) = |x - y|$$

5. Majorants et minorants

Définitions

  • $M$ est un majorant de $A \subset \mathbb{R}$ si $\forall x \in A, x \leq M$
  • $m$ est un minorant de $A \subset \mathbb{R}$ si $\forall x \in A, x \geq m$
  • Un ensemble est majoré s'il admet un majorant
  • Un ensemble est minoré s'il admet un minorant
  • Un ensemble est borné s'il est majoré et minoré

Borne supérieure et inférieure

  • La borne supérieure de $A$ est le plus petit majorant, notée $\sup(A)$
  • La borne inférieure de $A$ est le plus grand minorant, notée $\inf(A)$

6. Encadrements

Définition

Encadrer un réel $x$, c'est trouver deux réels $a$ et $b$ tels que $a \leq x \leq b$.

Abscisse d'un point

L'amplitude de l'encadrement est $b - a$.

Exercices

Exercice 1

Résoudre les inéquations suivantes :

  • $2x - 3 < x + 5$
  • $-2x + 1 \leq 3x - 4$
  • $\frac{x + 1}{x - 2} \geq 0$

Exercice 2

Simplifier :

  • $|3 - 5|$
  • $|x - 2|$ si $x \geq 2$
  • $|x - 2|$ si $x < 2$

Exercice 3

Résoudre dans $\mathbb{R}$ :

  • $|x - 3| = 5$
  • $|x + 2| < 4$
  • $|2x - 1| \geq 3$

Exercice 4

Montrer que pour tous $x, y \in \mathbb{R}$ : $|x| + |y| \geq |x + y|$.

Exercice 5

Encadrer $\sqrt{2}$ entre deux décimaux à $10^{-2}$ près.
