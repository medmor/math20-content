Notions de logique

I. Proposition – Fonction propositionnelle

Définition

Une proposition est un énoncé qui peut être vrai (V) ou faux (F), mais pas les deux à la fois.

Définition

Soit E un ensemble non vide. On appelle fonction propositionnelle (ou prédicat) d'une variable x de E, tout énoncé contenant la variable x, qui devient une proposition lorsque la variable x est remplacée par un élément de E.

Applications

Soient les fonctions propositionnelles suivantes :

A(x) : « x ∈ ℝ ; 3x² − 2x − 65 = 0 »

B(x) : « x ∈ ℝ ; 5x² − 2√15 x + 3 = 0 »

C(x) : « x ∈ ℝ ; x³ − 6x + 7 = 0 »

D(x) : « x ∈ ℝ ; −x² + x + 6 ≥ 0 »

P(x) : « x ∈ ℝ ; (2 sin x − 1) cos x = 0 »

Q(x) : « x ∈ [−π ; π] ; sin(x) cos(x) ≥ 0 »

II. Quantificateurs

Définition

Soit P(x) une fonction propositionnelle d'une variable x d'un ensemble non vide E.

À partir de la fonction propositionnelle « (x ∈ E) ; P(x) », on définit :

  • La proposition « (∃x ∈ E) ; P(x) » qui se lit « il existe au moins x ∈ E tel que P(x) », et qui est vraie lorsqu'il existe au moins x ∈ E vérifiant la propriété P(x). Le symbole ∃ s'appelle le quantificateur existentiel.
  • La proposition « (∀x ∈ E) ; P(x) » qui se lit « pour tout x ∈ E, P(x) » ou « quel que soit x ∈ E ; P(x) » et qui est vraie lorsque tous les éléments de E vérifient la propriété P(x). Le symbole ∀ s'appelle le quantificateur universel.

Remarques

  • Dans les apparences, « (∀x ∈ E) ; P(x) » ne dépend d'aucun x ! La lettre x figurant dans cette proposition a le statut de variable muette. En effet, cette proposition peut aussi être écrite : « (∀y ∈ E) ; P(y) », ou encore « (∀z ∈ E) ; P(z) », sans en modifier le sens.
  • Il en est de même de la proposition « (∃x ∈ E) ; P(x) » : elle affirme qu'il existe au moins un élément x ∈ E tel que P(x) soit vraie, mais n'en définit aucun en particulier.
  • Pour le quantificateur existentiel, on rencontre dans certains énoncés « (∃!x ∈ E) ; P(x) ». L'ajout du signe ! derrière le symbole ∃ le transforme et l'énoncé devient : « il existe un unique élément x ∈ E tel que P(x) soit vraie ».
  • Les quantificateurs doivent être placés avant l'assertion mathématique qu'ils quantifient.
  • L'emploi de quantificateurs en guise d'abréviation au milieu d'une phrase en français est exclu : ils ne doivent figurer que dans une phrase mathématique.
  • L'ordre d'écriture des quantificateurs est fondamental pour le sens d'une phrase formelle :
    • Quand deux quantificateurs existentiels se suivent, on peut les échanger sans changer le sens.
    • Quand deux quantificateurs universels se suivent, on peut les échanger sans changer le sens.
    • Quand on inverse l'ordre de deux quantificateurs différents, le sens change.

Applications

Déterminer la valeur de vérité de chacune des propositions suivantes :

Q₁ : « (∃x ∈ ℝ) ; x² + x + 1 = 0 »

Q₂ : « (∃x ∈ ℝ) ; x² + x + 1 > 0 »

Q₃ : « (∃x ∈ ℕ) ; √(x² + 3) = 2x − 4 »

Q₄ : « (∃x ∈ ℝ) (∀y ∈ ℝ) ; x ≤ y »

Q₅ : « (∃x ∈ ℝ*) (∀y ∈ ℤ*) ; xʸ = 1 »

Q₆ : « (∀x ∈ ℝ) (∃y ∈ ℝ) ; x ≤ y »

Q₇ : « (∀x ∈ ℝ) (∀y ∈ ℝ) ; x²y² > xy »

III. Opérations sur les propositions

3-1. Négation d'une proposition

Définition

La négation d'une proposition P est la proposition notée P̄ ou ¬P telle que la proposition P̄ est vraie si la proposition P est fausse et la proposition P̄ est fausse si la proposition P est vraie.

La négation est un connecteur logique unaire défini par la table de vérité :

PP̄
VF
FV

Proposition

Soit P(x) une fonction propositionnelle d'une variable x d'un ensemble non vide E.

La négation de la proposition « (∀x ∈ E) P(x) » est la proposition « (∃x ∈ E) P̄(x) ».

La négation de la proposition « (∃x ∈ E) P(x) » est la proposition « (∀x ∈ E) P̄(x) ».

Conséquence

Pour montrer que la proposition « (∀x ∈ E) P(x) » est fausse, il suffit de montrer que sa négation « (∃x ∈ E) P̄(x) » est vraie.

Ce type de raisonnement est appelé raisonnement par contre-exemple.

Applications

En utilisant un raisonnement par contre-exemple, montrer que les propositions suivantes sont fausses :

P₁ : « (∀x ∈ ℝ*) x + √x ≥ 2 » P₂ : « (∀x ∈ ℝ) (∀y ∈ ℝ) 2x − 4y ≠ 5 »
P₃ : « (∀x ∈ ℝ*) x + 1/x ≥ 2 » P₄ : « (∀x ∈ ℝ⁺) (∀y ∈ ℝ⁺) x + y ≥ √x + √y »
P₅ : « (∀x ∈ ]0;1[) 2x/(x²(1−x²)) < 1 » P₆ : « (∀α ∈ ]0;1[) (∀β ∈ ]0;1[) 1/α + 1/β < 1 − αβ »

3-2. Disjonction de deux propositions

Définition

La disjonction de deux propositions P et Q est la proposition qu'on note (P ou Q). Elle est fausse seulement si P et Q sont toutes les deux fausses.

La disjonction est un connecteur logique binaire défini par la table de vérité :

PQP ou Q
VVV
VFV
FVV
FFF

Remarques

Les propositions (P ou Q) et (Q ou P) ont le même sens. On dit que la disjonction est une opération commutative.

Les propositions [(P ou Q) ou R] et [P ou (Q ou R)] ont le même sens. On dit que la disjonction est une opération associative.

Applications

Soit P une proposition.

  1. Montrer que la proposition P̄ ou (P et P) est fausse.

Soit a et b deux réels de l'intervalle ]4 ; +∞[, et on considère les deux équations :

(E) : x² + ax + b = 0 et (F) : x² + bx + a = 0

Soit Δ₁ le discriminant de (E) et Δ₂ celui de (F), et les propositions :

P₁ : « Δ₁ ≥ 0 » et P₂ : « Δ₂ ≥ 0 »

  1. Montrer que la proposition (P₁ ou P₂) est vraie.

3-3. Conjonction de deux propositions

Définition

La conjonction de deux propositions P et Q est la proposition qu'on note (P et Q). Elle est vraie seulement si P et Q sont toutes les deux vraies.

La conjonction est un connecteur logique binaire défini par la table de vérité :

PQP et Q
VVV
VFF
FVF
FFF

Remarques

Les propositions (P et Q) et (Q et P) ont le même sens. On dit que la conjonction est une opération commutative.

Les propositions [(P et Q) et R] et [P et (Q et R)] ont le même sens. On dit que la conjonction est une opération associative.

Les propositions [P et (Q ou R)] et [(P et Q) ou (P et R)] ont le même sens. On dit que la conjonction est distributive par rapport à la disjonction.

Les propositions [P ou (Q et R)] et [(P ou Q) et (P ou R)] ont le même sens. On dit que la disjonction est distributive par rapport à la conjonction.

Applications

Soit P une proposition.

  1. Montrer que la proposition P̄ ou (P et P) est vraie.
  2. Déterminer les réels x et y tels que : (y = 3x − 1 et x² − x = 0)
  3. En utilisant la distributivité de la conjonction logique par rapport à la disjonction logique, résoudre dans ℝ les systèmes suivants :

    (S₁) : x² − y² = 0 et 2x² + y² = 3

    (S₂) : xy − x + y = 1 et 2x² − xy − y = 0

    (S₃) : 4x² − y² = 0 et (x − 2)(y − 6) = 0

  4. Soit x et y deux réels tels que x ≥ 1 et y ≥ 1. Montrer que : √(x − 1) + √(y − 1) ≤ xy
  5. Déterminer le domaine de définition de la fonction f définie par : f(x) = √(x² − 1) + x/(2x − 3)

3-4. Implication de deux propositions

Définition

L'implication d'une proposition P à une proposition Q est la proposition qu'on note (P ⇒ Q). Elle est fausse seulement si P est vraie et Q est fausse.

Les deux propositions (P ⇒ Q) et (P̄ ou Q) ont la même valeur de vérité.

L'implication est un connecteur logique binaire défini par la table de vérité :

PQP ⇒ Q
VVV
VFF
FVV
FFV

Applications

  1. Déterminer la valeur de vérité des propositions suivantes :
    • P₁ : « (13 est premier) ⇒ (56 est divisible par 13) »
    • P₂ : « (47 est premier) ⇒ (47 est impair) »
  2. Soit x et y deux réels non nuls. On considère P : « 2x + 4y = 1 » et Q : « 1/(x² + y²) ≤ 20 ». Montrer que P ⇒ Q.
  3. Soit a et b deux réels. Montrer que : (|a| < 1 et |b| < 1) ⇒ |a + b| < |1 + ab|
  4. Soit x un nombre réel. Montrer les implications suivantes :

    √(x² + 1) − x = 2 ⇒ √(x² + 1) + x = ½

    |x| + |x − 1| = x ⇒ x = 1

  5. Soit x, y et z trois réels dont l'un est nul et les deux autres sont de signes contraires. On suppose que les réels x, y et z vérifient les implications suivantes : x = 0 ⇒ y = 0 ; x > 0 ⇒ y < 0 ; y ≠ 0 ⇒ z > 0. Déterminer parmi ces nombres celui qui est nul puis celui qui est strictement positif.
  6. a- Montrer que : (∀a ∈ ]1 ; +∞[) a²/(a − 1) ≥ 4

    b- En déduire que pour tous a et b de ]1 ; +∞[ : a²/(b − 1) + b²/(a − 1) ≥ 8

  7. Soit (x, y) ∈ ℝ² tel que |x| ≤ 1 et |y| ≤ 1. Montrer que : √(1 − x²) + √(1 − y²) ≤ 2√(1 − ((x + y)/2)²)

3-5. Condition suffisante – Condition nécessaire

Proposition

Les deux propositions suivantes sont vraies :

[(∀x ∈ E) ; A(x) ⇒ B(x)] ⇒ [((∀x ∈ E) ; A(x)) ⇒ ((∀x ∈ E) ; B(x))]

[(∃x ∈ E)(∀y ∈ F) A(x ; y)] ⇒ [(∀y ∈ F)(∃x ∈ E) A(x ; y)]

3-6. Équivalence de deux propositions

Définition

L'équivalence de deux propositions P et Q est la proposition qu'on note (P ⇔ Q). Elle est vraie seulement si P et Q ont la même valeur de vérité ; c'est-à-dire toutes les deux sont simultanément vraies ou simultanément fausses.

Les deux propositions (P ⇔ Q) et [(P ⇒ Q) et (Q ⇒ P)] ont la même valeur de vérité.

L'équivalence est un connecteur logique binaire défini par la table de vérité :

PQP ⇔ Q
VVV
VFF
FVF
FFV

Remarques

La notation « P ⇔ Q » se lit « P est équivalent à Q » ou « P équivaut à Q » et correspond en français à la phrase « P si et seulement si Q ».

Les propositions (P ⇔ Q) et (Q ⇔ P) ont la même table de vérité. On dit que l'équivalence est une opération commutative.

On s'intéresse plus souvent aux équivalences vraies qu'aux fausses. En pratique, on écrira « P ⇔ Q » uniquement lorsqu'il s'agit d'une assertion vraie.

Les expressions « condition nécessaire et suffisante », « si et seulement si », « il faut et il suffit » signifient toutes « équivalent » ou encore « ⇔ ».

En pratique, dans une rédaction, on n'emploiera jamais les symboles ⇔ et ⇒. Le seul endroit où le symbole ⇔ est toléré, c'est dans les résolutions d'équations ou d'inéquations.

Proposition

Soit P, Q et R trois propositions. Alors :

[(P ⇔ Q) et (Q ⇔ R)] ⇒ (P ⇔ R)

On dit que l'équivalence est une relation transitive.

Généralisation

Soit P, R₁, R₂, ..., Rₙ et Q des propositions. Alors :

[(P ⇔ R₁) et (R₁ ⇔ R₂) et ... et (Rₙ ⇔ Q)] ⇒ (P ⇔ Q)

Pratiquement : pour montrer qu'une proposition P est vraie, on montre que les équivalences successives P ⇔ R₁ et R₁ ⇔ R₂ et ... et Rₙ ⇔ Q sont vraies et que la proposition Q est vraie. Par conséquent, la proposition P. Ce type de raisonnement s'appelle « raisonnement par équivalences successives ».

Applications

Soient a et b deux réels positifs.

  1. Montrer que : a/√(a² + 1) < b/√(b² + 1) ⇔ a < b
  2. Montrer que : a√(1 + a²) = b√(1 + b²) ⇔ a = b

Soit (x, y) ∈ ℝ² tel que x ≥ 1 et y ≥ 4.

  1. Montrer que : √(x − 1) + 2√(y − 4) = (x + y)/2 ⇔ (x = 2 et y = 8)

Soit (x, y) ∈ ℝ².

  1. Établir les équivalences suivantes :

    |x + y| ≤ √(x² + 1) + √(y² + 1) ⇔ |√(x² + 1) − √(y² + 1)| ≤ |x − y|

    (x + y)(x − y)² < 0 ⇔ (x + y)³ > 4(x³ + y³)

Soit x, y et z trois réels strictement positifs.

  1. Établir les équivalences suivantes :

    xy√z < xz√y < yz√x ⇔ x < y < z

    |x − y| < z < x + y ⇔ |(x² + y² − z²)/(xy)| < 2

Proposition

Les deux propositions suivantes sont vraies :

[(∀x ∈ E) A(x) et B(x)] ⇔ [((∀x ∈ E) A(x)) et ((∀x ∈ E) B(x))]

[(∃x ∈ E) A(x) ou B(x)] ⇔ [((∃x ∈ E) A(x)) ou ((∃x ∈ E) B(x))]

Remarques

La proposition [(∃x ∈ E) A(x) et B(x)] ⇔ [((∃x ∈ E) A(x)) et ((∃x ∈ E) B(x))] n'est pas toujours vraie.

Exemple : considérons P : « ((∃x ∈ ℝ) cos x = 0) et ((∃x ∈ ℝ) sin x = 0) » et Q : « ((∃x ∈ ℝ) cos x = 0 et sin x = 0) ».

P est vraie car 0 est un réel tel que sin x = 0 et π/2 est un réel tel que cos x = 0. Q est fausse car pour tout x ∈ ℝ : cos²x + sin²x = 1 ≠ 0.

Méthodes de démonstration d'une équivalence

  • Raisonnement par double implication : On suppose que P est vraie et on montre que Q est vraie, et réciproquement. La démonstration se fait donc en deux étapes.
  • Raisonnement par équivalences successives : On procède en une seule étape. On passe de P à Q en utilisant à chaque fois des équivalences. Cette méthode est plus courte mais peut être plus fastidieuse puisqu'on doit vérifier que chaque enchaînement logique est bien une équivalence et pas seulement une implication. On lui préférera en général le raisonnement par double implication.

IV. Lois logiques et raisonnements

4-1. Loi logique ou tautologie

Définition

Soit Q₁, Q₂, ..., Qₙ des propositions. On appelle loi logique ou tautologie toute proposition P résultante de l'assemblage par des connecteurs logiques de propositions prises parmi Q₁, Q₂, ..., Qₙ qui est vraie quelle que soit la valeur de vérité des propositions en jeu.

Applications

Soit P et Q deux propositions.

  1. Montrer que les propositions suivantes sont des lois logiques :

    P ⇒ (Q ⇒ P)

    P ⇒ (P̄ ⇒ Q)

    (Q̄ ou P) ou (Q ou P̄)

    (P ⇔ Q) ⇔ [(P et Q) ou (P̄ et Q̄)]

4-2. Lois de Morgan

Proposition

Soit P et Q deux propositions. Les deux propositions suivantes sont des lois logiques :

(P et Q)̄ ⇔ (P̄ ou Q̄)

(P ou Q)̄ ⇔ (P̄ et Q̄)

Applications

Déterminer la négation des propositions suivantes :

P : « (∃x ∈ ℝ) 0 ≤ x < 1 »

Q : « (∀x ∈ ℝ) (x² = 1 ⇒ x = 1) »

R : « (∀a ∈ ℝ) (|a + 1| ≤ 2 ⇒ a ≥ −3) »

S : « (∀x ∈ ℝ) (∀y ∈ ℝ⁺) (x² ≤ y² ⇔ −y ≤ x ≤ y) »

4-3. Raisonnement par contraposée

Proposition

Soit P et Q deux propositions. L'implication Q̄ ⇒ P̄ s'appelle la contraposée de l'implication P ⇒ Q.

La contraposée d'une implication est équivalente à celle-ci :

(P ⇒ Q) ⇔ (Q̄ ⇒ P̄)

Remarques

Il faut bien distinguer entre la négation, la contraposée et la réciproque :

  • La négation de P ⇒ Q est (P et Q̄).
  • La réciproque de P ⇒ Q est Q ⇒ P.
  • La contraposée de P ⇒ Q est Q̄ ⇒ P̄.

Le recours au raisonnement par contraposée n'est pertinent que si cette contraposée est plus facile à prouver que l'implication directe.

Applications

En utilisant le raisonnement par contraposée, montrer les implications suivantes :

  1. (∀x ∈ [−1 ; +∞[) (x ≠ 0 ⇒ √(1 + x) ≠ 1 + x/2)
  2. (∀x ∈ ℝ) (x² ≠ 3 ⇒ 2/√(1 + x²) ≠ 1)
  3. (∀(x, y) ∈ ℝ²) (4y ≠ −3x ⇒ x − y ≠ 7(x + y))
  4. (∀(x, y) ∈ ℝ²) ((xy − 1)(x − y) ≠ 0 ⇒ x(y² + y + 1) ≠ y(x² + x + 1))
  5. (∀(x, y) ∈ [1 ; +∞[²) (x ≠ y ⇒ x² − 2x ≠ y² − 2y)
  6. (∀x ∈ ℝ) (x ∉ [−1 ; 4] ⇒ x² − 3x − 4 > 0)
  7. (∀x ∈ ℝ⁺) (x ≠ 0 ⇒ 1/(1 + √x) ≠ 1 − √x)
  8. (∀(x, y) ∈ ℝ²) (x² + y² ≤ 1 ⇒ |x + y| ≤ 2)
  9. (∀(x, y) ∈ ℝ²) (|2x² + 5xy + 3y²| ≤ 3 ⇒ (|x + y| ≤ 3 ou |2x + 3y| ≤ √3))
  10. (∀(x, y) ∈ ℝ²) (x² + xy + y² ≤ 3 ⇒ (|x + 2y| ≤ 2√3 et |x| ≤ 2))

4-4. Raisonnement par l'absurde

Proposition

Soit P et Q deux propositions. La proposition suivante est une loi logique :

[(P̄ ⇒ Q) et (P̄ ⇒ Q̄)] ⇒ P

Applications

  1. Soit a, b et c des réels positifs tels que ab < c. Montrer que : a < √c ou b < √c
  2. Soit f une fonction numérique définie sur ℝ telle que pour tout (x, y) ∈ (ℝ*)² : f(xy) = f(x)f(y). On suppose que f(1) ≠ 0. Montrer que : (∀x ∈ ℝ*) f(x) ≠ 0
  3. Montrer par l'absurde que : (∀n ∈ ℕ) √(4n + 2) ∉ ℕ

4-5. Raisonnement par disjonction des cas

Proposition

Soit P et Q deux propositions. La proposition suivante est une loi logique :

[(P ⇒ R) et (Q ⇒ R)] ⇔ [(P ou Q) ⇒ R]

Applications

  1. Résoudre dans ℝ les inéquations suivantes :
    • (I₁) |x² − 4| − x² > 0
    • (I₂) |2x − 1| + |2x + 1| + |x| ≥ 4
    • (I₃) √(3 − x) + x < 0
  2. Montrer que le produit de trois entiers relatifs consécutifs est divisible par 6.
  3. Montrer que pour tout x ∈ ℝ :
    • a- x + √(x² + 1) > 0
    • b- |x| + |x + 1| + |x − 1| ≠ 0
    • c- |x − 1| ≤ x² − 2x + 2
  4. En utilisant un raisonnement par disjonction des cas, montrer les deux implications suivantes :

    (|a| ≤ c et |b| ≤ c) ⇒ |a + b| + |a − b| ≤ 2c

    |a + b| + |a − 2b| ≤ 3c ⇒ (|a| ≤ 2c et |b| ≤ c)

  5. Soit n un entier naturel. Démontrer que si n est impair, alors il s'écrit sous la forme n = 4k + 1 ou n = 4k + 3 avec k ∈ ℕ.
  6. Déduire que si l'entier n² − 1 n'est pas divisible par 8, alors l'entier n est pair.

4-6. Raisonnement par récurrence

Proposition

Soit P(n) une fonction propositionnelle qui dépend d'un entier naturel n et n₀ ∈ ℕ.

Si la proposition P(n₀) est vraie et si l'implication « P(n) ⇒ P(n + 1) » est vraie pour tout n ≥ n₀, alors la proposition P(n) est vraie pour tout entier n ≥ n₀.

Applications

  1. Montrer par récurrence que pour tout n ∈ ℕ* :

    1 + 2 + ... + n = n(n + 1)/2

    1³ + 2³ + ... + n³ = n²(n + 1)²/4

    1 × 2 + 2 × 3 + ... + n(n + 1) = n(n + 1)(n + 2)/3

  2. Montrer par récurrence que pour tout n ∈ ℕ* : 5²ⁿ⁺¹ + 2ⁿ⁺⁴ + 2ⁿ⁺¹ est divisible par 23.
  3. Soit a et b deux réels distincts et strictement positifs. Montrer par récurrence que : (∀n ∈ ℕ) ((a + b)/2)ⁿ ≤ (aⁿ + bⁿ)/2
