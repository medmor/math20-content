Trigonométrie

I. Formules de transformation de base

1-1. Formules d'addition

Proposition

Soient a et b deux réels quelconques.

  • cos(a + b) = cos a cos b − sin a sin b
  • cos(a − b) = cos a cos b + sin a sin b
  • sin(a + b) = sin a cos b + cos a sin b
  • sin(a − b) = sin a cos b − cos a sin b

Proposition

Soit a et b deux nombres réels tels que a ≠ π/2 + kπ et b ≠ π/2 + kπ avec k ∈ ℤ.

  • Si a − b ≠ π/2 + kπ avec k ∈ ℤ, alors : tan(a − b) = (tan a − tan b)/(1 + tan a tan b)
  • Si a + b ≠ π/2 + kπ avec k ∈ ℤ, alors : tan(a + b) = (tan a + tan b)/(1 − tan a tan b)

Proposition

Soit a et b deux réels non nuls tels que a² + b² ≠ 0.

Il existe un réel α tel que :

a cos x + b sin x = √(a² + b²) cos(x − α)

avec cos α = a/√(a² + b²) et sin α = b/√(a² + b²).

Applications

  1. Mettre les expressions suivantes sous forme de ρ cos(ωx + φ) :

    ① cos x − √3 sin x = 2 sin(π/6 − x)

    ② cos x + sin x = √2 cos(x − π/4)

    ③ √3 cos x + sin x = 2 sin(x + π/3)

    ④ −√6 cos x − √2 sin x = 2√2 cos(x + 5π/6)

  2. En remarquant que 5π/12 = 7π/12 − π/6, calculer tan(5π/12).
  3. Soit x ∈ ℝ tel que x ≠ π/2 + kπ et x ≠ π/4 + kπ et x ≠ −π/4 + kπ. Simplifier tan(π/4 − x) tan(π/4 + x).

1-2. Transformation de cos(2a), sin(2a) et tan(2a) – Formules de linéarisation

Proposition

Soit a et b deux nombres réels.

  • cos(2a) = cos²a − sin²a = 2cos²a − 1 = 1 − 2sin²a
  • sin(2a) = 2sin a cos a
  • cos²a = (1 + cos(2a))/2 et sin²a = (1 − cos(2a))/2 et tan²a = (1 − cos(2a))/(1 + cos(2a))
  • Si a ≠ π/2 + kπ et 2a ≠ π/2 + kπ pour tout k ∈ ℤ, alors : tan(2a) = 2tan a/(1 − tan²a)

Remarques

On a pour tout réel a :

1 + cos a = 2cos²(a/2) ; 1 − cos a = 2sin²(a/2) ; sin a = 2cos(a/2)sin(a/2)

On rappelle que pour tout réel a : sin a = cos(π/2 − a) = −cos(π/2 + a)

Applications

Soit x ∈ ℝ.

  1. Établir les égalités suivantes :

    ① 1 + sin x = (cos(x/2) + sin(x/2))²

    ② 1 + cos x + 2sin²(x/2) = 2

    ③ 2sin x + sin(2x) = 8sin(x/2)cos³(x/2)

    ④ 1 − cos x + sin x = 2sin(x/2)(sin(x/2) + cos(x/2))

    ⑤ 1 + cos x − sin x = 2√2 cos(x/2)cos(x/2 + π/4)

Soit α un réel tel que sin α ≠ −1.

  1. Montrer que : (1 − sin α)/(1 + sin α) = tan²(π/4 − α/2)

Soit x ∈ ℝ tel que x ≠ kπ pour tout k ∈ ℤ.

  1. Montrer que (1 − cos x)/sin x = tan(x/2) et sin x = (1 + cos x)tan(x/2).

1-3. Formules de cos(a), sin(a) et tan(a) en fonction de tan(a/2)

Proposition

Soit a un nombre réel tel que a ≠ π + 2kπ et a ≠ π/2 + kπ pour tout k ∈ ℤ.

On pose : t = tan(a/2).

On a : sin a = 2t/(1 + t²) et cos a = (1 − t²)/(1 + t²) et tan a = 2t/(1 − t²)

Applications

Soit x ∈ ℝ tel que tan x = 2.

  1. Calculer cos(2x), sin(2x) et tan(2x).

Soit x un nombre réel tel que x ≠ π + 2kπ pour tout k ∈ ℤ. On pose t = tan(x/2).

  1. Montrer que (1 − cos x)/(1 + cos x) = t² et sin x/(1 + cos x) = t.

Soit a et b deux réels de l'intervalle ]0 ; π/2[.

  1. Prouver les inégalités suivantes :

    ① 1 + 1/tan a < 1/tan(a/2)

    ② tan((a + b)/2) ≤ (tan a + tan b)/2

    (Indication : on pourra poser x = tan(a/2) et y = tan(b/2).)

II. Transformation de produits en sommes

Proposition

Soit a et b deux nombres réels. On a les formules suivantes :

  • cos a cos b = ½[cos(a + b) + cos(a − b)]
  • sin a sin b = ½[cos(a − b) − cos(a + b)]
  • sin a cos b = ½[sin(a + b) + sin(a − b)]
  • cos a sin b = ½[sin(a + b) − sin(a − b)]

Applications

  1. Écrire sous forme de sommes les produits suivants :

    A(x) = cos x cos(5x)

    B(x) = sin(3x) sin(4x)

    C(x) = cos(x − π/3) sin(2x + π/4)

III. Transformation de sommes en produits

Proposition

Soit p et q deux nombres réels. On a les formules suivantes :

  • cos p + cos q = 2cos((p + q)/2)cos((p − q)/2)
  • cos p − cos q = −2sin((p + q)/2)sin((p − q)/2)
  • sin p + sin q = 2sin((p + q)/2)cos((p − q)/2)
  • sin p − sin q = 2cos((p + q)/2)sin((p − q)/2)

Applications

  1. Écrire sous forme d'un produit les sommes suivantes :

    A(x) = sin x + sin(2x) + sin(3x)

    B(x) = 1 + cos x + cos(2x) + cos(3x)

    C(x) = 1 + sin(2x) − cos(2x)

    D(x) = sin x + sin(5x) + sin(7x)

  2. a- Écrire sous forme de produit l'expression cos x + cos(2x) où x ∈ ℝ.

    b- En déduire les solutions dans ℝ de l'équation : cos x + cos(2x) = 0

    c- Résoudre dans l'intervalle [0 ; π] l'inéquation : cos x + cos(2x) ≥ 0

  3. a- Factoriser l'expression sin x + sin(2x) + sin(3x) + sin(4x) où x ∈ ℝ.

    b- Résoudre dans l'intervalle [−π ; 0] l'inéquation : sin x + sin(2x) + sin(3x) + sin(4x) < 0

  4. a- Montrer que pour tout x ∈ ℝ :

    sin(x − π/3) + sin(3x) = 2sin(2x − π/6)cos(x + π/6)

    b- Résoudre dans ℝ l'équation : sin x + 2sin(3x) − √3 cos x = 0

    c- Résoudre dans l'intervalle [0 ; π/2[ l'inéquation : sin x + 2sin(3x) − √3 cos x ≤ 0

IV. Transformation de l'expression a cos(x) + b sin(x)

Proposition

Soit a et b deux nombres réels tels que (a ; b) ≠ (0 ; 0).

Alors il existe un réel α tel que :

a cos x + b sin x = √(a² + b²) cos(x − α)

avec cos α = a/√(a² + b²) et sin α = b/√(a² + b²).

Applications

  1. Mettre les expressions suivantes sous forme de ρ cos(ωx + φ) :

    A(x) = cos(2x) + sin(2x)

    B(x) = cos x − √3 sin x

    C(x) = 3cos(x/2) − √3 sin(x/2)

  2. a- Montrer que : (∀x ∈ ℝ) cos(2x) − √3 sin(2x) = cos(2x + π/3)

    b- En déduire les solutions dans ℝ de l'équation : cos(2x) − √3 sin(2x) = 1

    c- Résoudre dans l'intervalle ]−π/2 ; π/2] l'inéquation : cos(2x) − √3 sin(2x) ≥ 1

  3. Résoudre dans l'intervalle ]−π ; 2π[ l'inéquation : cos(x/2) + sin(x/2) ≥ −1
