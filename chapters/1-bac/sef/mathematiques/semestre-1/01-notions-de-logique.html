Notions de logique

I. Définitions

1-1. Proposition

Une proposition est un énoncé mathématique qui peut être soit vrai (V), soit faux (F), mais jamais les deux à la fois.

Exemples :

  • « 2 + 2 = 4 » est une proposition vraie.
  • « Paris est la capitale de l'Allemagne » est une proposition fausse.
  • « x > 5 » n'est pas une proposition (car sa valeur de vérité dépend de x).

1-2. Fonction propositionnelle

Une fonction propositionnelle (ou prédicat) est un énoncé contenant une ou plusieurs variables, qui devient une proposition lorsque les variables sont fixées.

Exemple : « x > 5 » est une fonction propositionnelle. Si x = 7, elle devient la proposition vraie « 7 > 5 ». Si x = 3, elle devient la proposition fausse « 3 > 5 ».

1-3. Quantificateurs

Quantificateur universel (∀) : ∀x ∈ E, P(x) signifie « pour tout élément x de l'ensemble E, la propriété P(x) est vraie ».

Quantificateur existentiel (∃) : ∃x ∈ E, P(x) signifie « il existe au moins un élément x dans E tel que P(x) est vraie ».

Exemples :

  • ∀x ∈ ℝ, x² ≥ 0 — vraie.
  • ∃x ∈ ℝ, x² = 2 — vraie (car √2 ∈ ℝ).
  • ∃x ∈ ℝ, x² = −1 — fausse (dans ℝ).

Négation des quantificateurs :

  • ¬(∀x ∈ E, P(x)) ≡ ∃x ∈ E, ¬P(x)
  • ¬(∃x ∈ E, P(x)) ≡ ∀x ∈ E, ¬P(x)

Exemple : La négation de « Tous les élèves ont réussi » est « Il existe au moins un élève qui n'a pas réussi ».

II. Opérations sur les propositions

2-1. La négation d'une proposition

La négation d'une proposition P, notée ¬P (lire « non P »), est vraie lorsque P est fausse, et fausse lorsque P est vraie.

2-2. La conjonction et la disjonction de deux propositions

Conjonction (∧) : P ∧ Q (lire « P et Q ») est vraie si et seulement si P et Q sont toutes deux vraies.

Disjonction (∨) : P ∨ Q (lire « P ou Q ») est vraie si au moins l'une des deux propositions est vraie.

2-3. L'implication de deux propositions

L'implication P ⇒ Q (lire « P implique Q » ou « si P alors Q ») est fausse uniquement lorsque P est vraie et Q est fausse.

PQP ⇒ Q

FVV

On dit que P est une condition suffisante pour Q, et que Q est une condition nécessaire pour P.

Remarque : L'implication P ⇒ Q ne suppose aucune relation causale entre P et Q. Elle exprime simplement que si P est vraie, alors Q doit aussi l'être.

La contraposée de P ⇒ Q est ¬Q ⇒ ¬P. Elle est logiquement équivalente à l'implication :

P ⇒ Q ≡ ¬Q ⇒ ¬P

La réciproque de P ⇒ Q est Q ⇒ P. Elle n'a pas nécessairement la même valeur de vérité que l'implication.

Exemple : Soit P : « x > 3 » et Q : « x > 0 ». Alors P ⇒ Q est vraie, mais Q ⇒ P est fausse (contre-exemple : x = 1).

2-4. L'équivalence de deux propositions

L'équivalence P ⇔ Q (lire « P si et seulement si Q ») est vraie lorsque P et Q ont la même valeur de vérité.

P ⇔ Q est vraie lorsque (P ⇒ Q) et (Q ⇒ P) sont toutes deux vraies.

2-5. Les lois logiques

Lois de De Morgan :

  • ¬(P ∧ Q) ≡ ¬P ∨ ¬Q
  • ¬(P ∨ Q) ≡ ¬P ∧ ¬Q

Autres lois :

  • ¬(¬P) ≡ P (double négation)
  • P ⇒ Q ≡ ¬P ∨ Q
  • P ⇔ Q ≡ (P ⇒ Q) ∧ (Q ⇒ P)

III. Types de raisonnements

3-1. Raisonnement par contre-exemple

Pour montrer qu'une proposition universelle « ∀x ∈ E, P(x) » est fausse, il suffit de trouver un élément de E pour lequel P(x) est fausse. Cet élément est appelé contre-exemple.

Exemple : La proposition « ∀x ∈ ℝ, x² > 0 » est fausse car x = 0 est un contre-exemple (0² = 0 et non > 0).

3-2. Raisonnement par des équivalences successives

Pour démontrer une équivalence P ⇔ Q, on peut enchaîner des équivalences logiques depuis P jusqu'à Q.

Exemple : Montrer que pour tout x ∈ ℝ : x² = 4 ⇔ x = 2 ou x = −2.

x² = 4 ⇔ x² − 4 = 0 ⇔ (x − 2)(x + 2) = 0 ⇔ x = 2 ou x = −2

3-3. Raisonnement déductif

On suppose que l'hypothèse P est vraie, et on démontre que la conclusion Q en découle. Cela prouve que P ⇒ Q.

Exemple : Démontrer que si n est pair, alors n² est pair.

Démonstration : Supposons n pair. Alors il existe k ∈ ℤ tel que n = 2k. Donc n² = (2k)² = 4k² = 2(2k²). Comme 2k² ∈ ℤ, n² est pair. □

3-4. Raisonnement par la contraposée

Pour démontrer P ⇒ Q, on démontre sa contraposée ¬Q ⇒ ¬P, qui lui est logiquement équivalente.

Exemple : Démontrer que si n² est impair, alors n est impair.

Démonstration : On montre la contraposée : si n est pair, alors n² est pair (déjà démontré ci-dessus). Donc si n² est impair, n ne peut pas être pair, donc n est impair. □

3-5. Raisonnement par la disjonction des cas

Pour démontrer une propriété P, on considère tous les cas possibles et on montre que dans chaque cas, P est vraie.

Exemple : Montrer que ∀n ∈ ℤ, n² ≥ 0.

Démonstration : Soit n ∈ ℤ. Deux cas :

  • Si n ≥ 0, alors n² ≥ 0.
  • Si n < 0, alors −n > 0, donc n² = (−n)² > 0 ≥ 0.

Dans tous les cas, n² ≥ 0. □

3-6. Raisonnement par l'absurde

Pour démontrer qu'une proposition P est vraie, on suppose que ¬P est vraie et on montre que cela conduit à une contradiction. Donc P doit être vraie.

Exemple : Démontrer que √2 est irrationnel.

Démonstration : Supposons par l'absurde que √2 est rationnel. Alors il existe p, q ∈ ℤ avec q ≠ 0 et p/q irréductible tel que √2 = p/q.

En élevant au carré : 2 = p²/q², donc p² = 2q². Ainsi p² est pair, donc p est pair (car le carré d'un impair est impair). Soit p = 2k.

Alors (2k)² = 2q², donc 4k² = 2q², donc q² = 2k². Ainsi q² est pair, donc q est pair.

Contradiction : p et q sont tous deux pairs, donc p/q n'est pas irréductible. Donc √2 est irrationnel. □

3-7. Raisonnement par la récurrence

Soit P(n) une propriété définie pour tout entier naturel n ≥ n₀. Pour démontrer que P(n) est vraie pour tout n ≥ n₀, on procède en deux étapes :

  1. Initialisation : Vérifier que P(n₀) est vraie.
  2. Hérédité : Supposer que P(k) est vraie pour un certain k ≥ n₀ (hypothèse de récurrence), et démontrer que P(k+1) est vraie.

Exemple : Démontrer que pour tout n ∈ ℕ*, 1 + 2 + ... + n = n(n+1)/2.

Démonstration : Soit P(n) : 1 + 2 + ... + n = n(n+1)/2.

  • Initialisation (n = 1) : 1 = 1 × 2/2 = 1. Vrai.
  • Hérédité : Supposons P(k) vraie : 1 + 2 + ... + k = k(k+1)/2.

Alors : 1 + 2 + ... + k + (k+1) = k(k+1)/2 + (k+1) = (k+1)(k/2 + 1) = (k+1)(k+2)/2.

Donc P(k+1) est vraie.

Par récurrence, P(n) est vraie pour tout n ∈ ℕ*. □

IV. Exercices

Exercice 1

Donner la négation des propositions suivantes :

  • a) ∀x ∈ ℝ, x² + 1 > 0
  • b) ∃x ∈ ℤ, x² = 2
  • c) ∀n ∈ ℕ, ∃m ∈ ℕ, m > n

Exercice 2

Montrer que l'implication « Si x² = 4, alors x = 2 » est fausse en donnant un contre-exemple.

Exercice 3

Démontrer par récurrence que pour tout n ∈ ℕ*, 1² + 2² + ... + n² = n(n+1)(2n+1)/6.

Exercice 4

Démontrer par l'absurde que si a et b sont des entiers tels que a² + b² est impair, alors a et b ne sont pas tous les deux pairs.
