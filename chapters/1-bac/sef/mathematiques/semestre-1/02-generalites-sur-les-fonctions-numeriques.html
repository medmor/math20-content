Généralités sur les fonctions numériques

I. Rappels

1-1. Fonction numérique

Une fonction numérique f est une relation qui à chaque élément x d'un ensemble de départ Df (appelé domaine de définition) associe un unique élément y d'un ensemble d'arrivée. On note : f : Df → ℝ et y = f(x).

Définitions :

  • Le domaine de définition Df est l'ensemble des valeurs de x pour lesquelles f(x) existe.
  • L'image de x par f est le nombre f(x).
  • Un antécédent de y par f est tout x ∈ Df tel que f(x) = y.
  • L'ensemble image Im(f) est l'ensemble de toutes les images f(x) lorsque x parcourt Df.

Règles pour déterminer Df :

  • Fonction polynôme : Df = ℝ
  • Fonction rationnelle : Df = ℝ \ {valeurs qui annulent le dénominateur}
  • Fonction racine carrée : Df = {x ∈ ℝ | expression sous la racine ≥ 0}

Exemple : Soit f(x) = 1/x. On a Df = ℝ \ {0}. L'image de 3 est f(3) = 1/3. Un antécédent de 5 est x = 1/5 car f(1/5) = 5.

Représentation graphique : La courbe représentative de f, notée Cf, est l'ensemble des points M(x ; f(x)) avec x ∈ Df.

1-2. Fonction paire – fonction impaire

Soit f une fonction définie sur un ensemble D symétrique par rapport à 0 (si x ∈ D, alors −x ∈ D).

Fonction paire : f est paire si ∀x ∈ D, f(−x) = f(x). Sa courbe Cf est symétrique par rapport à l'axe des ordonnées.

Fonction impaire : f est impaire si ∀x ∈ D, f(−x) = −f(x). Sa courbe Cf est symétrique par rapport à l'origine du repère.

Exemples :

  • f(x) = x² est paire car f(−x) = (−x)² = x² = f(x).
  • g(x) = x³ est impaire car g(−x) = (−x)³ = −x³ = −g(x).
  • h(x) = x + 1 n'est ni paire ni impaire : h(−x) = −x + 1 ≠ h(x) et ≠ −h(x).

1-3. Monotonie d'une fonction numérique

Soit f une fonction définie sur un intervalle I.

  • f est croissante sur I si pour tous x₁, x₂ ∈ I tels que x₁ ≤ x₂, on a f(x₁) ≤ f(x₂).
  • f est strictement croissante sur I si pour tous x₁, x₂ ∈ I tels que x₁ < x₂, on a f(x₁) < f(x₂).
  • f est décroissante sur I si pour tous x₁, x₂ ∈ I tels que x₁ ≤ x₂, on a f(x₁) ≥ f(x₂).
  • f est strictement décroissante sur I si pour tous x₁, x₂ ∈ I tels que x₁ < x₂, on a f(x₁) > f(x₂).

Exemples :

  • f(x) = x² est décroissante sur ]−∞ ; 0] et croissante sur [0 ; +∞[.
  • g(x) = 1/x est strictement décroissante sur ]−∞ ; 0[ et sur ]0 ; +∞[.

1-4. Taux d'accroissement d'une fonction f

Soit f une fonction définie sur un intervalle I et soient x₁, x₂ ∈ I avec x₁ ≠ x₂. Le taux d'accroissement (ou taux de variation) de f entre x₁ et x₂ est :

(f(x₂) − f(x₁)) / (x₂ − x₁)

Propriété :

  • f est croissante sur I si et seulement si pour tous x₁, x₂ ∈ I (x₁ ≠ x₂), (f(x₂) − f(x₁))/(x₂ − x₁) ≥ 0.
  • f est décroissante sur I si et seulement si pour tous x₁, x₂ ∈ I (x₁ ≠ x₂), (f(x₂) − f(x₁))/(x₂ − x₁) ≤ 0.

II. Fonction majorée – minorée – bornée

2-1. Définitions

Soit f une fonction définie sur un ensemble D.

  • f est majorée sur D s'il existe un réel M tel que pour tout x ∈ D, f(x) ≤ M. On dit que M est un majorant de f sur D.
  • f est minorée sur D s'il existe un réel m tel que pour tout x ∈ D, f(x) ≥ m. On dit que m est un minorant de f sur D.
  • f est bornée sur D si f est à la fois majorée et minorée sur D.

Exemples :

  • f(x) = sin(x) est bornée sur ℝ car −1 ≤ sin(x) ≤ 1 pour tout x ∈ ℝ.
  • g(x) = x² est minorée sur ℝ par 0, mais non majorée.

2-2. Extremums d'une fonction

Soit f une fonction définie sur un intervalle I.

  • f admet un maximum en a ∈ I si pour tout x ∈ I, f(x) ≤ f(a). Le nombre f(a) est le maximum de f sur I.
  • f admet un minimum en a ∈ I si pour tout x ∈ I, f(x) ≥ f(a). Le nombre f(a) est le minimum de f sur I.

Exemple : f(x) = x² admet un minimum en x = 0, qui vaut f(0) = 0.

2-3. Fonction périodique

Une fonction f définie sur Df est périodique de période T > 0 si :

  • Pour tout x ∈ Df, x + T ∈ Df
  • f(x + T) = f(x)

Exemple : La fonction sinus est périodique de période 2π car sin(x + 2π) = sin(x) pour tout x ∈ ℝ.

Propriété : Si f est périodique de période T, il suffit d'étudier f sur un intervalle de longueur T, puis de reproduire la courbe par translation de vecteur T·i⃗.

III. Comparaison de deux fonctions et interprétation géométrique

3-1. Égalité de deux fonctions

Deux fonctions f et g sont égales sur un ensemble D si :

  • Df = Dg = D
  • Pour tout x ∈ D, f(x) = g(x)

On note alors f = g sur D.

3-2. Comparaison de deux fonctions

Soient f et g deux fonctions définies sur un même ensemble D.

  • On dit que f ≤ g sur D si pour tout x ∈ D, f(x) ≤ g(x).
  • On dit que f ≥ g sur D si pour tout x ∈ D, f(x) ≥ g(x).

Interprétation géométrique :

  • f ≤ g sur D signifie que la courbe Cf est en dessous de (ou sur) la courbe Cg sur D.
  • f ≥ g sur D signifie que la courbe Cf est au-dessus de (ou sur) la courbe Cg sur D.

Exemple : Soient f(x) = x² et g(x) = x. Sur [0 ; 1], on a f(x) ≤ g(x) car x² ≤ x pour x ∈ [0 ; 1]. Sur [1 ; +∞[, on a f(x) ≥ g(x) car x² ≥ x pour x ≥ 1.

IV. Composée de deux fonctions

4-1. Vocabulaire

Soient f : A → B et g : B → C deux fonctions. La fonction composée de f suivie de g, notée g ∘ f, est la fonction qui à tout x ∈ A associe g(f(x)).

4-2. Définition

Soient f définie sur I et g définie sur J telles que f(I) ⊆ J. La composée g ∘ f est définie sur I par :

(g ∘ f)(x) = g(f(x))

Exemple : Soit f(x) = x + 1 et g(x) = x². Alors (g ∘ f)(x) = g(f(x)) = (x + 1)². Et (f ∘ g)(x) = f(g(x)) = x² + 1. On remarque que g ∘ f ≠ f ∘ g en général.

4-3. Monotonie de fonctions composées

Soient f définie sur I et g définie sur J avec f(I) ⊆ J.

f sur Ig sur Jg ∘ f sur I

Propriété sur la parité :

  • Si f et g sont paires, alors g ∘ f est paire.
  • Si f et g sont impaires, alors g ∘ f est impaire.
  • Si f est paire et g quelconque, alors g ∘ f est paire.

V. Étude et représentation graphique de certaines fonctions

5-1. Fonction f(x) = ax³ (a ≠ 0)

Propriétés :

  • Df = ℝ
  • f est impaire : f(−x) = a(−x)³ = −ax³ = −f(x)
  • Cf est symétrique par rapport à l'origine
  • f(0) = 0, donc Cf passe par l'origine

Sens de variation :

  • Si a > 0 : f est croissante sur ℝ
  • Si a < 0 : f est décroissante sur ℝ

Tableau de variation (a > 0) :

x−∞0+∞
f(x)−∞↗0↗+∞

Exemple : f(x) = 2x³. f est impaire, strictement croissante sur ℝ (car a = 2 > 0).

5-2. Fonction f(x) = √(x + a)

Domaine de définition : Df = [−a ; +∞[ (il faut que x + a ≥ 0).

Propriétés :

  • f(−a) = 0, donc Cf passe par le point (−a ; 0)
  • f est strictement croissante sur [−a ; +∞[
  • f n'est ni paire ni impaire (sauf si a = 0 et le domaine n'est pas symétrique)

Tableau de variation :

x−a+∞
f(x)0↗+∞

Exemple : f(x) = √(x − 2). Df = [2 ; +∞[. f(2) = 0. L'antécédent de 3 : √(x − 2) = 3 ⇒ x − 2 = 9 ⇒ x = 11.

VI. Exercices

Exercice 1

Soit la fonction f définie par f(x) = x² − 4x + 3.

  1. Déterminer Df.
  2. Calculer f(2) et les antécédents de 0.
  3. Étudier la parité de f.
  4. Étudier les variations de f.

Exercice 2

Soit f(x) = √(x + 1) et g(x) = x² − 1.

  1. Déterminer Df et Dg.
  2. Déterminer Dg∘f et exprimer (g ∘ f)(x).
  3. Étudier la parité de g ∘ f.
