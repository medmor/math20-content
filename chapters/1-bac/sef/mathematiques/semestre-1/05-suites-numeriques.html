Les suites numériques

I. Généralités sur les suites

1-1. Définition

Une suite numérique est une fonction définie sur ℕ (ou à partir d'un certain rang) à valeurs dans ℝ. On note une suite (un), où un désigne le terme de rang n.

Une suite peut être définie :

  • Par une formule explicite : un = f(n). Exemple : un = 2n + 1.
  • Par récurrence : on donne le premier terme et une relation un+1 = g(un). Exemple : u0 = 3 et un+1 = 2un − 1.

1-2. Représentation graphique

Dans un repère orthonormé, on place les points Mn(n, un). La représentation graphique d'une suite est un nuage de points dont les abscisses sont entières.

II. Suites majorée, minorée, bornée

2-1. Définitions

Soit (un) une suite numérique.

  • (un) est majorée s'il existe un réel M tel que pour tout n, un ≤ M.
  • (un) est minorée s'il existe un réel m tel que pour tout n, un ≥ m.
  • (un) est bornée si elle est à la fois majorée et minorée.

Exemple : La suite un = sin(n) est bornée car −1 ≤ sin(n) ≤ 1 pour tout n ∈ ℕ.

III. Monotonie d'une suite

3-1. Définitions

  • (un) est croissante si pour tout n, un+1 ≥ un.
  • (un) est décroissante si pour tout n, un+1 ≤ un.
  • (un) est strictement croissante si pour tout n, un+1 > un.
  • (un) est strictement décroissante si pour tout n, un+1 < un.
  • (un) est constante si pour tout n, un+1 = un.
  • (un) est monotone si elle est croissante ou décroissante.

3-2. Méthodes d'étude du sens de variation

  1. Étudier le signe de un+1 − un :
    • Si un+1 − un ≥ 0, alors (un) est croissante.
    • Si un+1 − un ≤ 0, alors (un) est décroissante.
  2. Comparer le quotient un+1/un à 1 (si tous les termes sont strictement positifs) :
    • Si un+1/un ≥ 1, alors (un) est croissante.
    • Si un+1/un ≤ 1, alors (un) est décroissante.
  3. Utiliser une fonction associée : si un = f(n) où f est définie sur [a, +∞[, le sens de variation de (un) est le même que celui de f sur [a, +∞[.

Exemple : Soit un = n² − 4n + 3. La fonction f(x) = x² − 4x + 3 a pour dérivée f'(x) = 2x − 4. Donc f est décroissante sur ]−∞, 2] et croissante sur [2, +∞[. Ainsi (un) est décroissante pour n ≤ 1 et croissante pour n ≥ 2.

3-3. Convergence et limites

On dit que (un) converge vers un réel ℓ si, pour tout intervalle ouvert contenant ℓ, tous les termes de la suite appartiennent à cet intervalle à partir d'un certain rang. On note : limn→+∞ un = ℓ.

Une suite qui ne converge pas est dite divergente.

Propriétés :

  • Si une suite converge, sa limite est unique.
  • Toute suite convergente est bornée.
  • Toute suite croissante et majorée converge.
  • Toute suite décroissante et minorée converge.

Opérations sur les limites : Soient lim un = ℓ et lim vn = ℓ'. Alors :

  • lim (un + vn) = ℓ + ℓ'
  • lim (un × vn) = ℓ × ℓ'
  • Si ℓ' ≠ 0, lim (un/vn) = ℓ/ℓ'
  • lim (k × un) = k × ℓ pour tout réel k

Théorème des gendarmes : Si un ≤ vn ≤ wn à partir d'un certain rang et lim un = lim wn = ℓ, alors lim vn = ℓ.

IV. Suites arithmétiques

4-1. Définition

Une suite (un) est arithmétique de raison r s'il existe un réel r tel que pour tout n ∈ ℕ :

un+1 = un + r

4-2. Propriétés

  • Terme général : un = u0 + n × r (si premier terme u0), ou un = u1 + (n−1) × r (si premier terme u1).
  • Somme des n+1 premiers termes (de u0 à un) : S = (n+1)(u0 + un) / 2
  • Somme des n premiers termes (de u1 à un) : S = n(u1 + un) / 2
  • Si r > 0, (un) est strictement croissante.
  • Si r < 0, (un) est strictement décroissante.
  • Si r = 0, (un) est constante.

Démonstration du terme général (par récurrence) :

  • Initialisation : pour n = 0, u0 = u0 + 0 × r. Vrai.
  • Hérédité : supposons uk = u0 + k × r. Alors uk+1 = uk + r = u0 + k × r + r = u0 + (k+1) × r. Vrai.

Donc pour tout n ∈ ℕ, un = u0 + n × r. □

Exemple : Soit u0 = 3 et un+1 = un + 5. C'est une suite arithmétique de raison 5. Terme général : un = 3 + 5n. Somme des 6 premiers termes (u0 à u5) : S = 6×(3+28)/2 = 93.

V. Suites géométriques

5-1. Définition

Une suite (un) est géométrique de raison q (q ≠ 0) s'il existe un réel q tel que pour tout n ∈ ℕ :

un+1 = q × un

5-2. Propriétés

  • Terme général : un = u0 × qn (si premier terme u0), ou un = u1 × qn−1 (si premier terme u1).
  • Somme des n+1 premiers termes (de u0 à un) : si q ≠ 1, S = u0 × (1 − qn+1) / (1 − q).
  • Si q = 1, la suite est constante et S = (n+1) × u0.
  • Si q > 1 et u0 > 0, (un) est strictement croissante.
  • Si 0 < q < 1 et u0 > 0, (un) est strictement décroissante.
  • Si q = 1, (un) est constante.
  • Si q < 0, (un) est alternée.

Démonstration du terme général (par récurrence) :

  • Initialisation : pour n = 0, u0 = u0 × q0 = u0. Vrai.
  • Hérédité : supposons uk = u0 × qk. Alors uk+1 = q × uk = q × u0 × qk = u0 × qk+1. Vrai.

Donc pour tout n ∈ ℕ, un = u0 × qn. □

Démonstration de la somme : Soit S = u0 + u0q + u0q² + ... + u0qn. On multiplie par (1−q) :

(1−q)S = u0(1 + q + q² + ... + qn) − u0(q + q² + ... + qn+1) = u0(1 − qn+1).

Donc S = u0(1 − qn+1)/(1 − q). □

Exemple : Soit u0 = 2 et un+1 = 3 × un. C'est une suite géométrique de raison 3. Terme général : un = 2 × 3n. Somme des 5 premiers termes : S = 2 × (1 − 35)/(1 − 3) = 2 × (−242)/(−2) = 242.

VI. Exercices

Exercice 1

Soit la suite (un) définie par un = 3n − 2. Montrer qu'elle est strictement croissante.

Exercice 2

Soit la suite (un) définie par un = 1/n pour n ∈ ℕ*. Montrer qu'elle est strictement décroissante et minorée par 0.

Exercice 3

Soit (un) une suite arithmétique de raison r = 4 et de premier terme u0 = 7. Calculer u20 et la somme S des 21 premiers termes.

Exercice 4

Soit (un) une suite géométrique de raison q = 1/3 et de premier terme u0 = 9. Calculer u5 et la somme des 6 premiers termes.

Exercice 5

Soit la suite (un) définie par u0 = 2 et un+1 = ½un + 1. Montrer que (un) est décroissante et minorée par 2. En déduire sa limite.

Exercice 6

En utilisant le théorème des gendarmes, montrer que limn→+∞ sin(n)/n = 0.
