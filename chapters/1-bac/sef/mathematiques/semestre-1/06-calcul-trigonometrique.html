Calcul trigonométrique

I. Formules d'addition et de soustraction

1-1. Formules du cosinus

Pour tous réels α et β :

  • cos(α + β) = cos α cos β − sin α sin β
  • cos(α − β) = cos α cos β + sin α sin β

1-2. Formules du sinus

Pour tous réels α et β :

  • sin(α + β) = sin α cos β + cos α sin β
  • sin(α − β) = sin α cos β − cos α sin β

1-3. Formules de la tangente

Pour tous réels α et β tels que cos α cos β ≠ 0 :

  • tan(α + β) = (tan α + tan β) / (1 − tan α tan β)
  • tan(α − β) = (tan α − tan β) / (1 + tan α tan β)

Exemple : Calculer cos(7π/12).

7π/12 = π/3 + π/4. Donc : cos(7π/12) = cos(π/3)cos(π/4) − sin(π/3)sin(π/4) = (1/2)(√2/2) − (√3/2)(√2/2) = (√2 − √6)/4.

II. Transformation de sommes en produits et réciproquement

2-1. Sommes en produits (factorisation)

Pour tous réels p et q :

  • cos p + cos q = 2 cos((p+q)/2) cos((p−q)/2)
  • cos p − cos q = −2 sin((p+q)/2) sin((p−q)/2)
  • sin p + sin q = 2 sin((p+q)/2) cos((p−q)/2)
  • sin p − sin q = 2 cos((p+q)/2) sin((p−q)/2)

2-2. Produits en sommes (linéarisation)

Pour tous réels a et b :

  • cos a cos b = ½[cos(a+b) + cos(a−b)]
  • sin a sin b = ½[cos(a−b) − cos(a+b)]
  • sin a cos b = ½[sin(a+b) + sin(a−b)]

Exemple : Linéariser cos²x.

cos²x = cos x × cos x = ½[cos(2x) + cos(0)] = ½[cos(2x) + 1] = (1 + cos(2x))/2.

III. Autres transformations

3-1. Formules de duplication

Pour tout réel α :

  • cos(2α) = cos²α − sin²α = 2cos²α − 1 = 1 − 2sin²α
  • sin(2α) = 2 sin α cos α
  • tan(2α) = 2tanα / (1 − tan²α), si tan²α ≠ 1

3-2. Formules de linéarisation

Pour tout réel α :

  • cos²α = (1 + cos(2α)) / 2
  • sin²α = (1 − cos(2α)) / 2

3-3. Expression a cos x + b sin x

Soient a et b deux réels non simultanément nuls. On peut écrire :

a cos x + b sin x = √(a² + b²) cos(x − φ)

où φ est défini par : cos φ = a/√(a² + b²) et sin φ = b/√(a² + b²).

Exemple : Écrire 3 cos x + 4 sin x sous la forme R cos(x − φ).

R = √(9+16) = 5, cos φ = 3/5, sin φ = 4/5. Donc 3 cos x + 4 sin x = 5 cos(x − φ) où φ = arctan(4/3).

3-4. Changement de variable (substitution de Weierstrass)

Pour résoudre certaines équations trigonométriques, on pose t = tan(x/2). Alors :

  • sin x = 2t / (1 + t²)
  • cos x = (1 − t²) / (1 + t²)
  • tan x = 2t / (1 − t²)

IV. Équations trigonométriques

4-1. Équation cos x = a

Soit a ∈ [−1, 1]. L'équation cos x = a admet pour solutions :

  • Si a > 1 ou a < −1 : aucune solution.
  • Si |a| ≤ 1 : x = ± arccos(a) + 2kπ, k ∈ ℤ.

Cas particuliers :

  • cos x = 1 ⇔ x = 2kπ, k ∈ ℤ
  • cos x = 0 ⇔ x = π/2 + kπ, k ∈ ℤ
  • cos x = −1 ⇔ x = π + 2kπ, k ∈ ℤ

4-2. Équation sin x = a

Soit a ∈ [−1, 1]. L'équation sin x = a admet pour solutions :

  • Si a > 1 ou a < −1 : aucune solution.
  • Si |a| ≤ 1 : x = arcsin(a) + 2kπ ou x = π − arcsin(a) + 2kπ, k ∈ ℤ.

Cas particuliers :

  • sin x = 1 ⇔ x = π/2 + 2kπ, k ∈ ℤ
  • sin x = 0 ⇔ x = kπ, k ∈ ℤ
  • sin x = −1 ⇔ x = −π/2 + 2kπ, k ∈ ℤ

4-3. Équation tan x = a

Pour tout réel a, l'équation tan x = a admet pour solutions :

  • x = arctan(a) + kπ, k ∈ ℤ

4-4. Exemples résolus

Exemple 1 : Résoudre cos x = 1/2.

x = ±π/3 + 2kπ, k ∈ ℤ.

Exemple 2 : Résoudre sin x = √2/2.

x = π/4 + 2kπ ou x = 3π/4 + 2kπ, k ∈ ℤ.

Exemple 3 : Résoudre 2cos x − 1 = 0.

cos x = 1/2 ⇒ x = ±π/3 + 2kπ, k ∈ ℤ.

Exemple 4 : Résoudre cos(2x) = 1/2 sur ]−π, π].

2x = ±π/3 + 2kπ ⇒ x = ±π/6 + kπ. Sur ]−π, π] : x ∈ {−5π/6, −π/6, π/6, 5π/6}.

V. Cercle trigonométrique

5-1. Définition

Le cercle trigonométrique est le cercle de rayon 1 centré à l'origine du repère orthonormé (O ; i⃗, j⃗). Un point M du cercle est repéré par un réel x (mesure en radians de l'angle orienté (i⃗, OM⃗)). Ses coordonnées sont (cos x, sin x).

5-2. Propriétés fondamentales

  • cos²x + sin²x = 1
  • −1 ≤ cos x ≤ 1 et −1 ≤ sin x ≤ 1
  • cos et sin sont 2π-périodiques : cos(x + 2π) = cos x et sin(x + 2π) = sin x
  • tan est π-périodique : tan(x + π) = tan x
  • cos est paire : cos(−x) = cos x
  • sin est impaire : sin(−x) = −sin x
  • tan est impaire : tan(−x) = −tan x

5-4. Formules liées au cercle trigonométrique

Angles associés :

  • cos(π − x) = −cos x ; sin(π − x) = sin x
  • cos(π + x) = −cos x ; sin(π + x) = −sin x
  • cos(π/2 − x) = sin x ; sin(π/2 − x) = cos x
  • cos(π/2 + x) = −sin x ; sin(π/2 + x) = cos x

Démonstration de cos²x + sin²x = 1 :

Soit M(cos x, sin x) un point du cercle trigonométrique. La distance OM = 1, donc OM² = cos²x + sin²x = 1² = 1. □
