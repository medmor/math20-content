Le barycentre dans le plan

Dans tout ce chapitre, on se place dans un plan muni d'un repère (O ; i⃗ , j⃗).

I. Barycentre de deux points pondérés

1-1. Définition

Soient A et B deux points du plan et α, β deux réels tels que α + β ≠ 0.

On appelle barycentre des points pondérés (A ; α) et (B ; β), le point unique G tel que :

α GA⃗ + β GB⃗ = 0⃗

On note : G = bar{(A ; α) ; (B ; β)}

Propriété : Le barycentre G existe et est unique.

1-2. Propriétés

Relation de Chasles : Soit G le barycentre des points pondérés (A ; α) et (B ; β). Pour tout point M du plan, on a :

α MA⃗ + β MB⃗ = (α + β) MG⃗

Démonstration : Pour tout point M, on a MA⃗ = MG⃗ + GA⃗ et MB⃗ = MG⃗ + GB⃗. Donc :

α MA⃗ + β MB⃗ = α(MG⃗ + GA⃗) + β(MG⃗ + GB⃗) = (α + β)MG⃗ + (α GA⃗ + β GB⃗) = (α + β)MG⃗ + 0⃗ = (α + β)MG⃗ □

Coordonnées du barycentre : Si A(xA ; yA) et B(xB ; yB), alors :

G( (αxA + βxB) / (α + β) ; (αyA + βyB) / (α + β) )

Position du barycentre :

Signe de αβPosition de G
αβ > 0 (même signe)G ∈ ]AB[ (entre A et B)
αβ < 0 (signes contraires)G ∉ [AB] (en dehors du segment)

Isobarycentre : Si α = β ≠ 0, alors G est le milieu du segment [AB]. On dit que G est l'isobarycentre de A et B.

Homogénéité : Le barycentre de (A ; α) et (B ; β) est le même que celui de (A ; kα) et (B ; kβ) pour tout réel k ≠ 0.

Cas particuliers :

  • Si α = 0, alors G = B.
  • Si β = 0, alors G = A.
  • Les points A, G et B sont toujours alignés.

Exemple : Soient A(1 ; 2) et B(4 ; −1). Déterminer les coordonnées du barycentre G des points pondérés (A ; 3) et (B ; 1).

Solution :

xG = (3×1 + 1×4) / (3+1) = 7/4

yG = (3×2 + 1×(−1)) / (3+1) = 5/4

Donc G(7/4 ; 5/4). De plus, αβ = 3×1 = 3 > 0, donc G ∈ ]AB[.

II. Barycentre de trois points pondérés

2-1. Définition

Soient A, B et C trois points du plan et α, β, γ trois réels tels que α + β + γ ≠ 0.

On appelle barycentre des points pondérés (A ; α), (B ; β) et (C ; γ), le point unique G tel que :

α GA⃗ + β GB⃗ + γ GC⃗ = 0⃗

On note : G = bar{(A ; α) ; (B ; β) ; (C ; γ)}

2-2. Propriétés

Relation de Chasles : Pour tout point M du plan, on a :

α MA⃗ + β MB⃗ + γ MC⃗ = (α + β + γ) MG⃗

Démonstration : On sait que α GA⃗ + β GB⃗ + γ GC⃗ = 0⃗. Or MA⃗ = MG⃗ + GA⃗, MB⃗ = MG⃗ + GB⃗, MC⃗ = MG⃗ + GC⃗. Donc :

α MA⃗ + β MB⃗ + γ MC⃗ = α(MG⃗ + GA⃗) + β(MG⃗ + GB⃗) + γ(MG⃗ + GC⃗) = (α + β + γ)MG⃗ + (α GA⃗ + β GB⃗ + γ GC⃗) = (α + β + γ)MG⃗ + 0⃗ = (α + β + γ)MG⃗ □

Coordonnées du barycentre : Si A(xA ; yA), B(xB ; yB), C(xC ; yC), alors :

G( (αxA + βxB + γxC) / (α + β + γ) ; (αyA + βyB + γyC) / (α + β + γ) )

Isobarycentre : Si α = β = γ ≠ 0, alors G est le centre de gravité du triangle ABC.

Exemple : Soient A(1 ; 2), B(3 ; 0), C(−1 ; 4). Déterminer les coordonnées du barycentre G de (A ; 1), (B ; 2), (C ; 3).

Solution :

xG = (1×1 + 2×3 + 3×(−1)) / (1+2+3) = (1+6−3)/6 = 4/6 = 2/3

yG = (1×2 + 2×0 + 3×4) / 6 = (2+0+12)/6 = 14/6 = 7/3

Donc G(2/3 ; 7/3)

III. Barycentre de quatre points pondérés

3-1. Définition

Soient A, B, C et D quatre points du plan et α, β, γ, δ quatre réels tels que α + β + γ + δ ≠ 0.

On appelle barycentre des points pondérés (A ; α), (B ; β), (C ; γ) et (D ; δ), le point unique G tel que :

α GA⃗ + β GB⃗ + γ GC⃗ + δ GD⃗ = 0⃗

3-2. Propriétés

Relation de Chasles : Pour tout point M du plan :

α MA⃗ + β MB⃗ + γ MC⃗ + δ MD⃗ = (α + β + γ + δ) MG⃗

Coordonnées du barycentre :

xG = (αxA + βxB + γxC + δxD) / (α + β + γ + δ)

yG = (αyA + βyB + γyC + δyD) / (α + β + γ + δ)

Associativité : Soit G le barycentre de (A ; α), (B ; β), (C ; γ), (D ; δ). Si H est le barycentre de (A ; α) et (B ; β), et K est le barycentre de (C ; γ) et (D ; δ), alors G est le barycentre de (H ; α + β) et (K ; γ + δ).

IV. Associativité du barycentre

Propriété (associativité) : Soient A, B, C trois points du plan, α, β, γ trois réels tels que α + β + γ ≠ 0. Soit G le barycentre de (A ; α), (B ; β) et (C ; γ). Si H est le barycentre de (A ; α) et (B ; β) (avec α + β ≠ 0), alors G est aussi le barycentre de (H ; α + β) et (C ; γ).

Démonstration :

Par définition de H : α HA⃗ + β HB⃗ = 0⃗

Par définition de G : α GA⃗ + β GB⃗ + γ GC⃗ = 0⃗

On veut montrer que (α + β) GH⃗ + γ GC⃗ = 0⃗.

On a : α GA⃗ + β GB⃗ + γ GC⃗ = α(GH⃗ + HA⃗) + β(GH⃗ + HB⃗) + γ GC⃗ = (α + β)GH⃗ + (α HA⃗ + β HB⃗) + γ GC⃗ = (α + β)GH⃗ + 0⃗ + γ GC⃗ = (α + β)GH⃗ + γ GC⃗ = 0⃗ □

Exemple : Soient A(1 ; 0), B(0 ; 2), C(3 ; 1). Déterminer les coordonnées de G, barycentre de (A ; 2), (B ; 1), (C ; 3) en utilisant l'associativité.

Solution : Soit H le barycentre de (A ; 2) et (B ; 1) :

xH = (2×1 + 1×0)/(2+1) = 2/3, yH = (2×0 + 1×2)/3 = 2/3. Donc H(2/3 ; 2/3).

G est le barycentre de (H ; 3) et (C ; 3), donc isobarycentre de H et C :

xG = (2/3 + 3)/2 = 11/6, yG = (2/3 + 1)/2 = 5/6. Donc G(11/6 ; 5/6).

V. Centre de gravité d'un triangle

Le centre de gravité d'un triangle ABC est l'isobarycentre de A, B et C, c'est-à-dire le point G, barycentre de (A ; 1), (B ; 1) et (C ; 1). Il vérifie :

GA⃗ + GB⃗ + GC⃗ = 0⃗

Propriétés :

  • Pour tout point M du plan : MA⃗ + MB⃗ + MC⃗ = 3 MG⃗
  • Le centre de gravité G est situé aux 2/3 de chaque médiane, à partir du sommet.
  • Les trois médianes d'un triangle sont concourantes en G.
  • Ses coordonnées sont : G( (xA + xB + xC)/3 ; (yA + yB + yC)/3 )

Exemple : Soient A(0 ; 0), B(4 ; 0), C(0 ; 6). Le centre de gravité est G(4/3 ; 2).

VI. Exercices

Exercice 1

Soient A(−2 ; 1), B(1 ; 3), C(4 ; 0). Calculer les coordonnées du barycentre G de (A ; 2), (B ; −1), (C ; 3).

Exercice 2

Soient A et B deux points distincts du plan. Montrer que le barycentre G de (A ; 2) et (B ; −1) appartient à la droite (AB) mais pas au segment [AB].

Exercice 3

Soient A, B, C trois points non alignés et G leur isobarycentre. Montrer que pour tout point M du plan : MA⃗ + MB⃗ + MC⃗ = 3 MG⃗.

Exercice 4

Soient A(1 ; 0), B(0 ; 2), C(3 ; 1). Déterminer les coordonnées du barycentre de (A ; 2), (B ; −1), (C ; 3) en utilisant l'associativité.

Exercice 5

Soit ABC un triangle. I est le milieu de [BC]. Montrer que le centre de gravité G du triangle ABC est le barycentre de (A ; 2) et (I ; 1).

Exercice 6

Soient A, B, C trois points du plan. Montrer que si G est le barycentre de (A ; 1), (B ; 2), (C ; 3), alors pour tout point M : MA⃗ + 2MB⃗ + 3MC⃗ = 6MG⃗.
