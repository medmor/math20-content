Les limites d'une fonction numérique

I. Limite finie en un point

1-1. Définition

Soit f une fonction définie sur un intervalle ouvert contenant a (sauf peut-être en a), et soit ℓ un réel. On dit que f(x) tend vers ℓ quand x tend vers a, et on note :

limx→a f(x) = ℓ

si pour tout ε > 0, il existe δ > 0 tel que :

0 < |x − a| < δ ⇒ |f(x) − ℓ| < ε

Exemple : Montrer que limx→2 (3x − 1) = 5.

Soit ε > 0. On a |(3x − 1) − 5| = |3x − 6| = 3|x − 2|. Il suffit de choisir δ = ε/3. Alors si 0 < |x − 2| < δ, on a |f(x) − 5| < ε. □

1-2. Unicité de la limite

Si f admet une limite en a, alors cette limite est unique.

II. Limite infinie en un point

2-1. Définition

On dit que f(x) tend vers +∞ (resp. −∞) quand x tend vers a si :

Pour tout A > 0, il existe δ > 0 tel que : 0 < |x − a| < δ ⇒ f(x) > A (resp. f(x) < −A).

On note : limx→a f(x) = +∞ (resp. −∞).

Exemple : Montrer que limx→0 1/x² = +∞.

Soit A > 0. On a 1/x² > A ⇔ x² < 1/A ⇔ |x| < 1/√A. On choisit δ = 1/√A. □

III. Limite en +∞ ou −∞

3-1. Limite finie en +∞

Soit f définie sur ]b, +∞[. On dit que limx→+∞ f(x) = ℓ si :

Pour tout ε > 0, il existe B > 0 tel que : x > B ⇒ |f(x) − ℓ| < ε.

On définit de manière analogue limx→−∞ f(x) = ℓ.

3-2. Limite infinie en +∞

On dit que limx→+∞ f(x) = +∞ si :

Pour tout A > 0, il existe B > 0 tel que : x > B ⇒ f(x) > A.

On définit de manière analogue les limites −∞ en +∞, et les limites en −∞.

Exemple : Montrer que limx→+∞ 1/x = 0.

Soit ε > 0. On a |1/x| < ε ⇔ x > 1/ε. On choisit B = 1/ε. □

IV. Limites à gauche et à droite

Limite à droite : limx→a⁺ f(x) = ℓ si x tend vers a par valeurs supérieures à a.

Limite à gauche : limx→a⁻ f(x) = ℓ si x tend vers a par valeurs inférieures à a.

Théorème : limx→a f(x) = ℓ ⇔ limx→a⁻ f(x) = limx→a⁺ f(x) = ℓ.

Exemple : Étudier limx→0 1/x.

On a limx→0⁺ 1/x = +∞ et limx→0⁻ 1/x = −∞. Les limites à gauche et à droite sont différentes, donc limx→0 1/x n'existe pas.

V. Propriétés des limites

5-1. Opérations sur les limites

Soient f et g admettant des limites finies en a : lim f(x) = ℓ et lim g(x) = ℓ'.

  • lim [f(x) + g(x)] = ℓ + ℓ'
  • lim [f(x) × g(x)] = ℓ × ℓ'
  • Si ℓ' ≠ 0, lim [f(x)/g(x)] = ℓ/ℓ'
  • lim [k × f(x)] = k × ℓ, pour tout réel k

Ces règles s'étendent aux limites infinies avec les conventions usuelles (+∞ + +∞ = +∞, etc.), sauf dans les cas de formes indéterminées.

5-2. Formes indéterminées

Les formes suivantes ne permettent pas de conclure directement :

  • +∞ − ∞
  • 0 × ∞
  • ∞ / ∞
  • 0 / 0

Il faut transformer l'expression (factorisation, division par le terme dominant, etc.).

5-3. Théorème des gendarmes

Soient f, g, h trois fonctions telles que g(x) ≤ f(x) ≤ h(x) au voisinage de a. Si limx→a g(x) = limx→a h(x) = ℓ, alors limx→a f(x) = ℓ.

Exemple : Calculer limx→0 x sin(1/x).

On a −1 ≤ sin(1/x) ≤ 1, donc −|x| ≤ x sin(1/x) ≤ |x|. Or limx→0 (−|x|) = limx→0 |x| = 0. Par le théorème des gendarmes, limx→0 x sin(1/x) = 0.

5-4. Théorème de comparaison

Si f(x) ≥ g(x) au voisinage de a et limx→a g(x) = +∞, alors limx→a f(x) = +∞.

Si f(x) ≤ g(x) au voisinage de a et limx→a g(x) = −∞, alors limx→a f(x) = −∞.

5-5. Théorème de composition

Si limx→a f(x) = b et limy→b g(y) = L (avec g continue en b), alors limx→a g(f(x)) = L.

5-6. Limites usuelles

  • limx→0 sin(x)/x = 1
  • limx→0 (1 − cos(x))/x = 0
  • limx→0 (1 − cos(x))/x² = 1/2
  • limx→+∞ xn = +∞ (n ∈ ℕ*)
  • limx→+∞ 1/xn = 0 (n ∈ ℕ*)
  • limx→0⁺ 1/x = +∞ ; limx→0⁻ 1/x = −∞

VI. Exemples résolus

Exemple 1 : Calculer limx→2 (x² − 4)/(x − 2).

Forme indéterminée 0/0. On factorise : (x² − 4)/(x − 2) = (x − 2)(x + 2)/(x − 2) = x + 2 pour x ≠ 2. Donc lim = 4.

Exemple 2 : Calculer limx→+∞ (3x² + 2x − 1)/(x² + 5).

Forme ∞/∞. On divise par x² : = (3 + 2/x − 1/x²)/(1 + 5/x²) → 3/1 = 3.

Exemple 3 : Calculer limx→+∞ (2x + 1)/(x − 3).

On divise par x : = (2 + 1/x)/(1 − 3/x) → 2/1 = 2.

Exemple 4 : Calculer limx→1 (x³ − 1)/(x − 1).

Forme 0/0. On factorise : x³ − 1 = (x − 1)(x² + x + 1). Donc lim = 1² + 1 + 1 = 3.

Exemple 5 : Calculer limx→0 sin(x)/x.

On a cos x ≤ sin(x)/x ≤ 1 pour x ∈ ]−π/2, π/2[ \ {0}. Comme limx→0 cos x = 1, par le théorème des gendarmes, limx→0 sin(x)/x = 1.

VII. Asymptotes

7-1. Asymptote horizontale

La droite d'équation y = ℓ est asymptote horizontale à Cf en +∞ (resp. −∞) si limx→+∞ f(x) = ℓ (resp. limx→−∞ f(x) = ℓ).

7-2. Asymptote verticale

La droite d'équation x = a est asymptote verticale à Cf si limx→a⁺ f(x) = ±∞ ou limx→a⁻ f(x) = ±∞.

7-3. Asymptote oblique

La droite d'équation y = ax + b est asymptote oblique à Cf en +∞ (resp. −∞) si :

limx→+∞ [f(x) − (ax + b)] = 0

où a = limx→+∞ f(x)/x et b = limx→+∞ [f(x) − ax].

Exemple : Soit f(x) = (x² + 1)/(x − 1). Déterminer les asymptotes.

  • Asymptote verticale : x = 1 car limx→1 f(x) = ±∞.
  • Asymptote oblique : f(x) = x + 1 + 2/(x − 1). Donc limx→±∞ [f(x) − (x + 1)] = limx→±∞ 2/(x − 1) = 0. Asymptote oblique : y = x + 1.

VIII. Limites et continuité

Une fonction f est continue en a si et seulement si :

limx→a f(x) = f(a)

Propriétés :

  • Les fonctions polynômes sont continues sur ℝ.
  • Les fonctions rationnelles sont continues sur leur domaine de définition.
  • La composée de deux fonctions continues est continue.
  • Si f est continue en a et g est continue en f(a), alors g ∘ f est continue en a.

Exercice 3
